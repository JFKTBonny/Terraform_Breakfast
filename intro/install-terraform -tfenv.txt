Use tfenv to Manage Multiple Versions (Recommended!)

If you switch versions often:

# Install tfenv (Linux | macOS)

        git clone https://github.com/tfutils/tfenv ~/.tfenv
        echo 'export PATH="$HOME/.tfenv/bin:$PATH"' >> ~/.bashrc
        source ~/.bashrc

# Install latest Terraform

        tfenv install latest
        tfenv use latest


Now switching versions becomes simple:

        tfenv list
        tfenv use 1.8.3
